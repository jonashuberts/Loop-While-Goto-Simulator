Zulässige Ausdrücke

  • xi := xj + c Konstantenaddition
  • xi := xj -· c Monus-Subtraktion
  • xi := xj Zuweisung (xj + 0)

Ablaufsteuerung

  • P1; P2 Anweisungssequenz
  • LOOP xi DO P END Wiederhole P genau xi-mal (festgesetzt bei Eintritt)
Semantik: Register starten standardmäßig bei 0, außer definierte Eingaben. Das Ergebnis steht am Ende in x0. Die modifizierte Subtraktion (Monus) deckelt bei 0: xi := max(0, xj - c).

Zulässige Ausdrücke

  • xi := xj + c Konstantenaddition
  • xi := xj -· c Monus-Subtraktion
  • xi := xj Zuweisung (xj + 0)

Ablaufsteuerung

  • P1; P2 Anweisungssequenz
  • WHILE xi DO P END Wiederhole P, solange xi != 0 ist (dynamische Prüfung)
Semantik: WHILE-Programme können in Endlosschleifen laufen, falls die Schleifenbedingung nie 0 erreicht. Die restliche Variablen- und Subtraktionssemantik entspricht den LOOP-Programmen.

Instruktionen

  • Mi : xi := xj + c Zuweisung mit Label
  • Mi : xi := xj -· c Monus-Subtraktion mit Label
  • Mi : xi := xj Direkte Registerzuweisung

Sprungbefehle

  • Mi : GOTO Mk Unbedingter Sprung zu Label Mk
  • Mi : IF xj = c GOTO Mk Bedingter Sprung zu Mk, falls xj gleich c ist
Anforderung: Jede Zeile eines GOTO-Programms muss mit einem eindeutigen Label Mi: beginnen. Jedes angesprungene Ziel-Label muss existieren.
